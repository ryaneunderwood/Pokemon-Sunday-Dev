custom handlers must be created. Here's the expected v21 handler structure:

For Type-Power Boosters:
  Battle::ItemEffects::DamageCalcFromUser.add(:MAGNIFYINGGLASS, 
    proc { |item, user, target, move, mults, baseDmg, type|
      mults[:power_multiplier] *= 1.1 if type == :LIGHT
    }
  )

For Stat Boosters:
  Battle::ItemEffects::OnSwitchIn.add(:LEGENDITE,
    proc { |item, battler, battle|
      # Implementation needed - boost stats significantly
    }
  )

For Memory Items:
  Requires form change implementation via:
  - MultipleForms registration
  - GameData::ItemEffects for type changes
  - Compatibility with specific species

For Berry Items:
  Battle::ItemEffects::OnEndOfRound or similar for damage/status mitigation
